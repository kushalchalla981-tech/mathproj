About PCA Compressor

What is PCA?

Principal Component Analysis (PCA) is a statistical technique that transforms data into a new coordinate system where the greatest variance lies on the first coordinate (principal component). By keeping only the most significant components, we can compress data while preserving the most important information.

Technical Details

  • Per-Channel Mode: Processes each RGB channel independently using 1D PCA
  • Joint-Channel Mode: Treats RGB as 3D vectors for color-aware compression
  • Orientation Correction: Uses PCA on pixel distribution to detect rotation
  • Quality Metrics: Computes SSIM and PSNR for objective quality assessment
