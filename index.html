UNDERSTANDING PCA

How It Works

1. Center Data

Subtract the mean from each data point. This shifts the origin to the center of our data cloud.

x' = x - μ

2. Find Variance

Compute the covariance matrix to understand how pixels vary and relate to each other.

C = cov ( X )

3. Principal Axes

Find eigenvectors — the directions of maximum variance — and their eigenvalues.

THEORY

On the Mathematics of Dimensionality

The Method

Principal Component Analysis transforms data into an orthogonal coordinate system where the greatest variance by any projection lies on the first principal component. By projecting onto fewer dimensions, we achieve compression while preserving maximum information content.

01

Center Data

Subtract the mean from all observations

02

Compute Covariance

Calculate the covariance matrix C

03

Eigen-Decompose

Find eigenvalues λᵢ and eigenvectors vᵢ

04

Project

Project data onto k largest eigenvectors

05

Reconstruct

Build approximation from reduced data

Quality Assessment

We employ Structural Similarity (SSIM) and Peak Signal-to-Noise Ratio (PSNR) metrics. SSIM measures perceptual quality by comparing structural luminance, contrast, and texture patterns, while PSNR provides a mathematical assessment of reconstruction fidelity.
